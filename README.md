Git reminders

- Create local repository on PC using git init
- Configure remote repository using URL. Now known as origin.
- Use git add to stage changes in local repository. Choose desired file/files.
- Use git commit to commit changes.
- Add tag to branch using git tag <tag_name>
- Push tag to remote repo using git push origin main
- Create new branch using git branch <branch_name>
- Merge main branch with specified branch using git merge <branch_name>
- Push new main to remote repo.
